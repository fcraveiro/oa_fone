## Getting started

With Dart:

   $ dart pub add oa_fone

With Flutter:

   $ flutter pub add oa_fone


add a line like this to your package's pubspec.yaml :

environment:

dependencies:
  flutter:
    sdk: flutter
  oa_fone: ^1.0.1

dev_dependencies:
